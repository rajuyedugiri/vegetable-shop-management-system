# Vegetable Shop Management System

# Description
The Vegetable Shop Management System is a mini project developed using Visual Studio Code to simplify the management of a vegetable shop.

# Features
# Shopkeeper
1. Add new vegetables
2. Update vegetable details
3. Delete vegetables
4. View available stock
5. Access all sales and billing reports

# Customer
1. View available vegetables
2. Purchase multiple vegetables in a single order
3. Generate a single bill with the total amount

# Objectives
1. Simplify vegetable shop management.
2. Reduce manual work in maintaining inventory.
3. Generate accurate bills for customers.
4. Maintain sales records for future reference.

# Technologies Used
* Visual Studio Code
